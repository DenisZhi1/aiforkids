# Чек-лист публикации

- [ ] Проверить права на все встроенные изображения и персонажа.
- [ ] При необходимости выбрать и добавить файл `LICENSE`.
- [ ] Создать новый репозиторий GitHub.
- [ ] Загрузить содержимое этой папки в корень ветки `main`.
- [ ] Включить GitHub Pages: `Settings` → `Pages` → `Deploy from a branch` → `main` → `/(root)`.
- [ ] Открыть опубликованный адрес и проверить меню, звук и все четыре уровня.
- [ ] Проверить управление мышью, клавиатурой и касанием на мобильном устройстве.
- [ ] Добавить в описание репозитория ссылку на опубликованную игру.
