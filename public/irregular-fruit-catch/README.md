# Irregular Fruit Catch

Небольшая браузерная игра-тренажёр по неправильным английским глаголам. Игрок ловит правильные формы глаголов, набирает серии ответов и проходит четыре уровня с разной механикой.

Игра работает как обычная статическая страница: сборка, сервер и сторонние библиотеки не нужны.

## Запуск

Откройте файл `index.html` в современном браузере.

## Управление

- `←` / `→` или `A` / `D` — движение корзины.
- Мышь или касание — управление по горизонтали.
- `P` — пауза.
- На уровне 4 игрок направляет падающее слово в одну из корзин `V1`, `V2` или `V3`.

## Уровни

1. **Warm-up** — форма V2 в Past Simple, спокойная скорость.
2. **Speed Run** — форма V2, ускоренное падение фруктов.
3. **Tricky Forms** — нужно поймать V2 и не перепутать её с V1 или V3.
4. **Three Baskets** — сортировка слов по формам V1, V2 и V3.

Все четыре уровня доступны сразу.

## Публикация через GitHub Pages

1. Создайте на GitHub новый репозиторий.
2. Загрузите в корень репозитория все файлы из этой папки.
3. Откройте `Settings` → `Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/(root)`, затем нажмите `Save`.

Проект уже содержит `index.html` и `.nojekyll`, поэтому отдельная сборка не требуется.

## Техническая информация

- HTML5 Canvas, CSS и JavaScript без фреймворков.
- 4 игровых уровня и 76 неправильных глаголов.
- 41 изображение встроено непосредственно в `index.html`.
- Звуки правильного и неправильного ответа создаются программно.
- Лучший результат и настройка звука сохраняются только в браузере через `localStorage`.
- Игра не отправляет данные и не загружает внешние ресурсы.

## Проверка перед выпуском

Перед упаковкой были проверены синтаксис JavaScript, генерация вариантов ответа, все четыре уровня, правильные и неправильные попадания, управление, повторение ошибочных слов и завершение финального уровня.

## Права на материалы

Лицензия намеренно не добавлена. Перед публикацией публичного репозитория убедитесь, что у вас есть право распространять все встроенные фоны, фрукты, изображения корзины и персонажа. В игре сохранена видимая подпись автора исходных материалов.
