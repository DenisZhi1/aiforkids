# Bee Collector

**Bee Collector** — полностью автономная обучающая HTML5-игра для детей 5–12 лет. Игрок помогает пчёлке собирать английские слова, превращает успехи в золотой мёд и постепенно развивает собственный улей.

Игра работает в браузере без библиотек, сервера, регистрации и сетевых запросов. Графика, интерфейс, музыка и игровые данные встроены в один файл `index.html`.

## Возможности

- 42 английских слова с переводами, определениями и озвучиванием через браузер.
- 5 учебных режимов: Spelling, Listening, Picture, Translation и Definition.
- 5 тем: Starter Mix, Animals, Food, Transport и School.
- Комбо, бонусы скорости, награды, коллекция баночек и развитие улья.
- Собственная мультяшная графика, анимации, частицы и две музыкальные темы.
- Сохранение прогресса и настроек только в локальном хранилище браузера.
- Адаптивный интерфейс для компьютеров, планшетов и телефонов.
- Управление мышью, касанием и клавиатурой; поддержка системного режима уменьшенной анимации.

## Как играть

1. Нажмите **PLAY** и выберите учебный режим.
2. Найдите цветок с нужной следующей буквой.
3. Соберите слово по порядку до окончания времени.
4. Получите мёд, откройте новую награду и развивайте улей.

В Listening можно повторно нажать кнопку динамика, чтобы услышать слово. Кнопка звука и ползунок управляют общей громкостью. Клавиша `Escape` ставит игру на паузу или закрывает открытое окно.

## Запуск

Самый простой способ — открыть `index.html` в современном браузере.

Для локального веб-сервера можно выполнить:

```bash
python -m http.server 8000
```

Затем открыть `http://localhost:8000`.

## Проверка качества

В проект входит автономный аудит без сторонних пакетов:

```bash
npm run check
```

Он проверяет структуру страницы, синтаксис JavaScript, встроенные изображения и музыку, учебные данные, доступность, адаптивность, сохранения и отсутствие внешних зависимостей.

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub и загрузите в него содержимое этой папки.
2. Откройте **Settings → Pages**.
3. В разделе **Build and deployment** выберите **Deploy from a branch**.
4. Выберите ветку `main`, папку `/ (root)` и нажмите **Save**.

Файл `.nojekyll` добавлен, поэтому GitHub Pages будет отдавать проект как обычный статический сайт.

## Конфиденциальность

Игра не использует аналитику, рекламу, cookies или внешние API. Прогресс хранится локально на устройстве. Единственная внешняя ссылка — кнопка автора.
