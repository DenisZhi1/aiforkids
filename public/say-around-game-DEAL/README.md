# Say Around!

Автономная HTML-игра для разминки на уроках английского.

## Что находится в папке

- `index.html` — вся игра в одном файле: разметка, оформление, карточки и игровая логика.
- Никаких библиотек, серверов или дополнительных файлов не требуется.
- Игра работает локально: откройте `index.html` двойным щелчком.

## Как разместить на GitHub Pages

1. Создайте репозиторий GitHub.
2. Загрузите `index.html` в корень репозитория.
3. В настройках репозитория включите GitHub Pages для основной ветки и корневой папки.
4. GitHub покажет готовую публичную ссылку.
