Как играть

  1. Разделитесь на команды.В каждом раунде выберите одного объясняющего игрока.
  2. Объясняйте только по-английски.Нельзя произносить само слово, его часть или перевод.
  3. Угадывайте до сигнала.Правильный ответ приносит 1 балл, пропуск — 0 баллов.
  4. Передавайте ход.Побеждает команда, первой набравшая выбранное число баллов.
Совет учителю: для младших детей начните с уровня A1 и раундов по 60–90 секунд.
