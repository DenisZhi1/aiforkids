# Grammar Millionaire Junior

Готовая HTML-игра для уроков английского по механике quiz show: 15 вопросов, 4 варианта ответа, призовая лестница, 50/50, голосование класса, совет маскота, звуковые сигналы, конфетти и финальный экран.

При каждом запуске игра процедурно собирает новый набор заданий по тем же 15 темам. Меняются имена, предметы, животные, места, действия, правильные варианты и расположение ответов A-D.

Визуальный стиль: отдельное главное меню, оригинальная сцена Grammar Galaxy, welcome-маскот, маскот Professor Spark внутри игры, стеклянные игровые панели, неоновая подсветка, адаптивная верстка для компьютера и мобильного экрана.

Внизу главного меню добавлена авторская плашка со ссылкой: `https://vk.com/deandal`.

Темы подобраны для детей 7-9 лет, которые проходят Spotlight:

- `to be`
- `a / an`
- множественное число
- `have got`
- притяжательные местоимения
- `can / can't`
- `there is / there are`
- предлоги места
- Present Simple
- `do / does`
- Present Continuous
- `some / any`
- `was / were`
- `going to`
- comparatives

Откройте `index.html` в браузере. Генераторы вопросов находятся в `script.js`, их можно расширять своими словами и шаблонами для конкретного модуля Spotlight.
